Introducción

En esta sección se mostrara todo lo que se realizo para la creación del proyecto final, para tal efecto, se divide en tres segmentos la sustentación del proyecto, una primera parte que mostrara la estructura del proyecto, la segunda que mostrara el backend y finalmente en la ultima parte se mostrara el fronted del proyecto.

1.0 Estructura del proyecto

Este proyecto se basa en una metodología que aprovecha el motor de plantillas Jinja2, permitiendo la creación dinámica y eficiente de contenido HTML en el frontend a partir de datos proporcionados desde el backend. Las herramientas empleadas en el desarrollo incluyen tecnologías clave para garantizar un flujo de trabajo optimizado y una experiencia de usuario interactiva y adaptable; las herramientas empleadas en el proyecto son:

  1. Python: Como lenguaje base para el backend, brindando capacidad de procesamiento y manejo de lógica de negocio.
  2. Flask: Framework web minimalista para gestionar rutas, solicitudes HTTP y la integración entre el backend y las vistas en HTML.
  3. HTML, CSS y JavaScript: Tecnologías de frontend para estructurar, diseñar e interactuar con los elementos visuales de la aplicación.
  4. Bootstrap: Esta biblioteca de diseño front-end facilitó la creación de una interfaz responsiva, estéticamente atractiva y compatible con múltiples dispositivos, optimizando el diseño sin comprometer el rendimiento.
  5. VS Code: VS Code fue el entorno de desarrollo principal, ofreciendo una experiencia optimizada con extensiones específicas para Python, Flask y herramientas de depuración, lo cual mejoró la eficiencia en el desarrollo y mantuvo la calidad del código.

2.0 Backend

Luego de tener clara la estructura del proyecto, se entrara en el codigo o backend del proyecto, donde se encontrara el codigo que posteriormente se enlazara con html para que pase verse en consola a verse en una interfaz grafica.

Framework Flask

Para enlazar el backend con el el fronted, utilizamos el framework flask , en este caso no llamamos todo el framework, solo importamos lo que requerimos para el proyecto, por eso como se ve acontinuacion especificamos lo que importaremos de flask como se ve en la linea de codigo.

Codigo de los modulos

Los modulos son los 4 ejercicios tomados de parciales anteriores, cada modulo es un ejercicio y para que fuera posible mostrar en pantalla y no en consola, se tuvo que modificar el codigo, especificamente los print que se utilizaban para imprimir por consola, se debieron modificar por medio de return, para que pudiese mostrarse en la interfaz grafica como se muestra en las siguientes imagenes del codigo de cada modulo.

Modulo #1 (Contar Vocales)

En el primer modulo se tiene que realizar dos cosas, la primera comparar si las cadenas son iguales y almacenar el resultado en una lista para poder devolverlo al html y segundo generar la funcion para contar vocales dado el caso que ambas cadenas de texto sean iguales, podemos ver como el print se elimina y se usa el return.

Modulo #2 (Buscar Sub Cadena)

En este segundo modulo se realiza la verificacion de dos cadenas donde determinamos cual de las dos es larga y cual es mas corta, dada esa verificacion , evaluamos si la cadena corta esta en la larga que nos retorne el mensaje de que la corta es subcadena de la cadena larga, si se da lo contrario que nos retorne que la cadena corta no es subcadena de la cadena larga.

Modulo #3 (Buscar Palindromas)

En el tercer modulo se determina los numeros de espacios que puedan existir en un parrafo, se separa el parrafo en palabras, volviendolas subcadenas para identificar si existen palabras palindromas y determinar cuantas existen, por ultimo en el codigo le pedimos que nos recorra la lista y nos diga cuales son las palabras palindromas que existen en la lista donde se almacenaron y las muestre en pantalla, si no hay palindroma le pedimos al codigo que muestre el mensaje "No se encontraron palabras palindromas".

Modulo #4 (Primera letra mayuscula)

En el ultimo modulo se realizo un codigo que permitiera colocar la prfimera letra en mayuscula de cada palabra, como se requiere que sean palabras para colocarle mayuscula, empezamos eliminando errores al verficar que el texto ingresado no fuera un numero, si lo es, nos arrojara como respuesta que lo que acaba de digitar son numeros, si no son numeros el programa separara la frase ingresada en subcadenas y colocara la primera letra en mayuscula con ayuda del .capitalize() y nos retornara la frase con las primeras letras de cada palabra en mayuscula.
